# はじめに
## Nayco とは
[![nayco](/doc/img/nayco.svg)](https://github.com/tac0x2a/nayco)

Nayco(内湖) は、オンプレミスのデータの収集・蓄積・可視化を素早く立ち上げるためのオールインワン環境です。
