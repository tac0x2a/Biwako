# Nayco

[![nayco](./doc/img/nayco.svg)](https://github.com/tac0x2a/nayco)

Nayco(内湖) is an all in one micro Data Lake for IoT like data.

Please see [Usage.md](/doc/Usage.md) or [Usage.ja.md](/doc/Usage.ja.md)

# How to deploy
## Deploy
```sh
$ docker-compose up -d
```

## Shutdown
```sh
$ docker-compose down
```

## Backup
```sh
$ ./backup.sh
